Dropbox Plugin Demo

The Dropbox plugin allows your app to use Dropbox as an OAuth provider. Additionally, you can upload files to the linked Dropbox account.

Important
Make sure you replace the APP_KEY and APP_SECRET parameters in the plugin settings.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS/Android - App package.

Connect Dropbox account

In case the user has the Dropbox app installed, the app will present the native Dropbox authentication UI. If not, it falls back to the web interface. The account remains linked until this app is uninstalled.

Save to Dropbox

Capture an image and upload it to the Apps/AppName/Saves folder of your Dropbox account.
